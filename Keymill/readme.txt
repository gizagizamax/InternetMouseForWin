■概要

キーボード入力の詳細情報を確認するためのソフトです。
入力したキーの仮想キーコードやスキャンコード等をリアルタイムに表示します。
情報は一覧表形式で記録され、CSVファイルに保存することも可能です。


■機能

・キーボード入力の詳細情報を表示（一覧表に記録）
  取得/表示する情報は以下の通りです。
  ・日時 … キー入力日時
  ・仮想キーコード … スキャンコードをキーボードドライバが変換した仮想コード（押されたキーの目的を識別するための値）
  ・スキャンコード … キーボードからCPUに送られるコード（キーボードの各キーに割り当てられたキー識別のための値）
  ・拡張キーフラグ … 拡張キーを判定するためのフラグ（押されたキーが拡張キーの場合にTrue、それ以外の場合にFalse）
  ・イベント挿入フラグ … イベント挿入キーを判定するためのフラグ（押されたキーがソフトからのキー入力の場合にTrue、それ以外の場合にFalse）
  ・コンテキストコード … コンテキストコードを判定するためのフラグ（同時にAltキーが押されていた場合にTrue、それ以外の場合にFalse）
  ・遷移状態フラグ … 遷移状態を判定するためのフラグ（キーアップの場合にTrue、キーダウンの場合にFalse）
  ・タイムスタンプ … Windows起動してからの経過時間（単位：ミリ秒、長整数値の最大値2147483647を超えると0に戻る）
  ・追加情報 … メッセージに関連するその他の追加情報
  ・仮想キーコード定数 … 仮想キーコードに対応する記号定数
  ・キー … 仮想キーコード定数に対応するキー

・記録した情報をCSVファイルで保存


■動作環境

本ソフトの動作には、Microsoft .NET Framework 2.0/3.0/3.5のいずれかが必要です。（Vista以降のWindowsには初めからインストールされています。）
エラーが出てソフトが起動できない場合は、Windows Updateまたは下記URLよりMicrosoft .NET Frameworkをインストールしてください。
http://www.microsoft.com/ja-jp/download/details.aspx?id=25150

動作確認は下記のOSで行っています。（下記以外のOSでも、Microsoft .NET Framework 2.0/3.0/3.5のいずれかがインストールされていれば動作すると思います。）
・Windows 2000
・Windows XP（32bit版）
・Windows 7（64bit版）
・Windows 8（64bit版）


■インストール

インストーラ版の場合は、ダウンロードしたファイルを実行してください。
セットアップが始まりますので、画面の指示に従ってインストールしてください。

アーカイブ版の場合は、アーカイブ内のファイルを適当なフォルダに解凍してください。
バージョンアップの際は、前バージョンのファイルに上書きしてください。


■アンインストール

インストーラ版でインストールした場合は、コントロールパネルからアンインストールしてください。

アーカイブ版でインストールした場合は、インストールフォルダ、および設定ファイル保存フォルダを削除してください。
設定ファイル保存フォルダは、エクスプローラのアドレス欄に「%appdata%\KT Software\Keymill」と入力し、Enterキーを押すと開くことができます。

レジストリは使用していません。


■使用方法

Keymill.exe（またはKeymill.exeへのショートカット）を実行すると、ソフトが起動します。
ソフト起動後、キーボードの適当なキーを押してください。そのキー入力の詳細情報が表示されます。
ツールバーのボタンから以下の機能を利用できます。

・「記録停止/記録開始」ボタン … 情報の記録の停止/開始
・「クリア」ボタン … 記録したデータをクリア
・「CSV保存」ボタン … 記録したデータをCSV形式でファイルに保存
・「列幅を自動調整」 … 一覧の全ての列の幅をセル内の文字列の幅に合わせる
・「オプション設定」ボタン … Keymillのオプション設定画面を表示
・「バージョン情報」ボタン … Keymillのバージョン情報画面を表示

以下の手順で、アプリケーション設定を初期化することができます。
1) シフトキーを押しながら本ソフトを起動する
2) 確認メッセージが表示されるので、「はい」を選択
