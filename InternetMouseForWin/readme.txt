---InternetMouse について---

スマホを使用してWindowsパソコンを遠隔操作できるようになります。

---特徴---

インターネット回線を使用するため簡単に始められます。
難しい設定は不要なので、どなたでも使用できます。

スマホからWindowsパソコンを操作しますが、
Windowsパソコンからスマホへは何もデータを送信しません。
そのため情報漏えいのリスク無く、安全に使用できます。

---使い方---

１．動作環境にあったInternetMouseForWinを実行してください。
InternetMouseForWin：https://play.google.com/store/apps/details?id=com.akblab.internetmouse

Windows10、8、7用がありますが、起動できればどれを使用していただいても同じです。
10、8用の方が少し快適に動作します。

２．InternetMouseForWinのIDに数字が入るのを確認してください。
  自分で変更することもできますが、他人と被る可能性があるため注意してください。

３．スマホアプリにIDを設定してください。
Android：https://play.google.com/store/apps/details?id=com.akblab.internetmouse
iPhone/iPad:申請中
ブラウザ:検討中(技術的には問題無し)

４．スマホアプリを操作するとWindowsパソコンが動きます。

---備考---

IDは他人と被らないように発行されますが、自分で変えた場合は被ることがあります。
被ってしまうと他人に操作されてしまう可能性があるため
PWDにパスワードを設定して防いでください。
メールアドレスや電話番号を設定すると被りにくいとは思います。

Windows10、8、7用が別れているのは
Windows標準で入っているライブラリのバージョンが違うため、それぞれ用意しただけです。
Windows Update等で.Net Frameworkのバージョンが上がっていれば
どれでも使用できるようになっているはずです。
以下、それぞれのバージョン
Windows10 - .Net Framework 4.6
Windows8  - .Net Framework 4.5
Windows7  - .Net Framework 3.5

一部、効かない組み合わせのキーがあります。
Alt+Tab等はWindowsの仕様上、アプリからは実行できないようです。

バグや要望があればブログに連絡ください。
http://ameblo.jp/gizagizamax/

元々はブラウザから操作できるよう作り始めたので
ブラウザ版も用意しようか検討中です。
レイアウト作るのが辛いため、そこだけがネックになってます。
